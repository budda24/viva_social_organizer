# VivaTech Social Organizer

Invite-only connector community for VivaTech Paris 2026. Hybrid Flutter Web + WhatsApp command bot.

**Source of truth:** [/home/franek/.claude/plans/becoming-the-organizer-of-spicy-charm.md](/home/franek/.claude/plans/becoming-the-organizer-of-spicy-charm.md)

## Stack

- Flutter Web (responsive) — member portal + admin
- Firebase: Identity Platform (LinkedIn OIDC), Firestore, Cloud Functions, Hosting (region `europe-west1`)
- Anthropic Claude (Haiku 4.5) via Claude Agent SDK — bot brain runs on the host's laptop
- Whapi.cloud — WhatsApp HTTP API (unofficial)

## Day 1 setup checklist (external)

These cannot be automated — you need to do them:

1. **LinkedIn Developer app** at [linkedin.com/developers/apps](https://www.linkedin.com/developers/apps) → create app → add "Sign In with LinkedIn using OpenID Connect" product → request access (usually fast).
2. **Firebase project** at [console.firebase.google.com](https://console.firebase.google.com) → create project → upgrade to **Identity Platform** (Authentication → Sign-in method → Add new provider → Custom OIDC). Add LinkedIn as a custom OIDC provider with the credentials from step 1.
3. **Whapi.cloud** account at [whapi.cloud](https://whapi.cloud) → create channel → link a **dedicated WhatsApp number** (not your personal one, use an eSIM). Save the API token and channel ID.
4. **Anthropic API key** at [console.anthropic.com](https://console.anthropic.com) → save in Firebase Functions Secrets Manager and locally in `.env` (never commit).
5. **Domain** — buy domain (`foundersatviva.com` or chosen name). Will configure Firebase Hosting custom domain later.

## Running locally

(filled in after Flutter + Firebase CLI install)

```bash
# Cloud Functions
cd functions && npm install && npm run build && firebase emulators:start

# Bot brain (separate terminal)
cd bot && npm install && npm run dev

# Flutter Web
flutter run -d chrome
```
